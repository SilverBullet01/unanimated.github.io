How to make xvid keyframes
by herkz


xvid keyframes are usually more accurate than x264 keyfarmes (which can be off by 1 frame near scene changes). However, they do behave slightly different so it may take a while for you to get used to it. Also, making xvid keyframes is only useful if you have no intention of making a workraw and you just want to time to the premux.

Things you need to install:

1. Avisynth

Install this first.

2. SCXviD.dll

Place this is the Avisynth\Plugins folder wherever you just installed it.

3. ffms2.dll

Copy this from you Aegisub folder and place it in the Avisynth\Plugins folder. The version included with the most recent build of Aegisub 3.0 should also have the most recent version of ffms2.dll.

4. avs2yuv.exe

If you're lazy like me you can just place this in your C:\Windows folder. If you know what you're doing you can put it in whatever folder and add it to your path. You could also put it in whatever folder you're going to put the batch file from later in.

5. This batch script (below if you don't want to download it)

@echo off
echo Making SCXvid keyframes...
set video=%~1
set video2=%~n1

echo FFvideosource("%video%") > "%video2%_keyframes.avs"
echo SCXvid("%video2%_keyframes.log") >> "%video2%_keyframes.avs"
avs2yuv "%video2%_keyframes.avs" -o NUL
del "%video2%_keyframes.avs"
del "%video%.ffindex"
echo Keyframes complete
@pause


Copy and paste this into a text file and rename it to keyframes.bat (or whatever.bat that you want). Personally, I place this on my desktop and drag the file I want to make keyframes for to the batch script. Although you can't see the progress, the script will tell you when it's done. It will also delete the files it created and leave you with just the keyframes. You can open these in Aegisub and time to the premux without spending time and hard drive space making a workraw.
