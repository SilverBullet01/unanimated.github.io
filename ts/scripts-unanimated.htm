"Cycle" type scripts:

Blur Cycle v1.7 - Adds \blur0.6, then cycles through 0.8, 1, 1.2, 1.5, 2, 3, 4, 5, 0.4, 0.5, back to 0.6.
Border Cycle v1.7 - Adds \bord0 to selected lines, then cycles through 1, 2, 3, 4, 5, 6, 7, 8, 9, back to 0
Shadow Cycle v1.7 - Adds \shad0 to selected lines, then cycles through 1, 2, 3, 4, 5, 6, 7, 8, 9, back to 0.
Alpha Cycle v1.7 - Adds \alpha&HFF& to selected lines, then cycles through 00, 30, 60, 80, A0, D0, FF

Cycles
v1.7 - All of the above in one. (Delete the 4, and reassign hotkeys.) Also cycles \an.

[bind them to hotkeys to make them really efficient. sequences can be modified.]


"On/Off" type scripts:

Italicize v1.6 - italicizes or unitalicizes text [based on style and tags]
Make Bold v1.6 - switches between bold and regular [based on style and tags]


Scripts with GUI - essential:

Script Cleanup v3.1 - removes comments and other unneeded stuff from selected lines
Blur and Glow v2.4 - creates layers with blur and/or glow
HYDRA v3.9 - adds tags, does transforms, plus a bunch of special functions
Hyperdimensional Relocator v3.2 - anything you need to do with pos, move, org, clip, and then some
Recalculator v2.1 - recalculates values of selected tags (multiply or add)
Colorize v4.2 - does all kinds of things with colours
Selectricks v2.8 - selects and sorts lines based on a bunch of criteria
Multi-line Editor v1.5 - allows you to edit multiple lines at once easily
MultiCopy v3.1 - copy various things from multiple lines to other multiple lines
Apply fade v3.8 - does fades including alpha or colour transforms and more
ShiftCut v2.7 - does things with timing. TPP functions and more.
Unimportant v2.7 - does a shitload of things that didn't fit into other scripts
Time signs from timecodes v2.6 - times signs from timecodes like {TS 5:36}
Change Capitalization v1.8 - lowercase, uppercase, capitalize lines or words


Multifunctions:

Masquerade v2.3 - mask, shift tags, alignment, mocha scale, alpha shift, alpha time, strikealpha
Copyfax This v2.8 - fax, copy stuff, copy tags, copy text, copy clip, copy colours, split by \N
Modify Text v1.1 - italicize, bold, capitalization, \an8, \q2, honorifix


Scripts without GUI:

Line Breaker v2.2 - inserts and shifts linebreaks
Join / Split v1.0 - joins lines / split lines (both text and timing)
Snap v1.2 - snaps to keyframes like TPP (but you can hotkey it)
Re-Split v1.1 - moves the last word of the line to the start of the next, or the other way round


Encoding/muxing scripts:

Encode - Hardsub v1.0 - encodes and hardsubs, whole video or clip
Multiplexer v1.1 - muxes loaded script + video, chapters, creates CRC + xdelta
